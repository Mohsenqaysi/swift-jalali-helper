# Jalali Calendar Helper
Written in Swift.

Please keep in mind in you're in OS X or IOS you should use `NSCalendarIdentifierPersian` instead.

## Example
```
let jh = jalaliHelper()
print(jh.toJalali()) // (1394, 10, 4)
print(jh.getMonth()) // دی
print(jh.getDay()) // جمعه
```
